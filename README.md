# 超级无敌简单选题系统

## Introduction - 介绍
2021年2018届滨江学院软件工程实训题目之选课系统
### Summary - 概要
超级无敌简单选题系统包含数据库脚本
### Features - 特性
基于Java&MySQL的命令行型选课系统
## Requirements - 必要条件
jdk 1.8以上，MySQL5以上
## Configuration - 配置
MySQL@5.7，mysql-connector-java-5.1.7，Java@1.8.0_271
